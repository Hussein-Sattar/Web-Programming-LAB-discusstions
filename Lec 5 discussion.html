1. What are the limitations of CSS ?

       1- Layout Complexity:
        CSS can struggle with complex layouts, often requiring additional tools or frameworks.

       2- No Variables/Constants:
        It lacks native support for variables and constants, hindering code maintainability.

       3- Global Scope:
        Styles have a global scope, leading to naming conflicts and unintended style overrides.

       4- Limited Dynamic Capabilities:
        CSS is declarative and lacks dynamic features, making responsive design more challenging.

       5- Cross-browser Compatibility:
        Different browsers may interpret CSS rules differently, causing rendering inconsistencies.

       6- Limited Expressiveness:
        CSS is not a programming language, limiting its ability to implement complex logic.

       7- Print Styling Challenges:
        Styling for print media can be challenging due to printer variations and settings.

       8- Performance Impact:
        Large stylesheets can impact page loading times and overall performance.

       9- Vertical Centering Challenges:
        Achieving vertical centering can be cumbersome, though Flexbox has improved this.

       10- No Inherent Logic Support:
        CSS lacks native support for programming logic, making dynamic styling scenarios challenging.
